# CV Analysointisovellus

Tämä on Pythonilla toteutettu CV-analysointityökalu, joka käyttää Flask-backendia ja Tkinter-GUI:ta.

## Toiminnallisuus

- Analysoi CV:n sanamäärän
- Tunnistaa avainsanoja CV:stä ja työpaikkailmoituksesta
- Laskee niiden vastaavuuden prosentteina
- Tekee sentimenttianalyysin CV-tekstistä

## Rakenne

- `backend/` sisältää Flask-palvelimen ja analyysiluokan
- `frontend/` sisältää graafisen käyttöliittymän ja testiklientin
- `README.md` ohjeistaa käytön
- `requirements.txt` sisältää riippuvuudet
